Regression 1: a 1 unit increase in the democracy index is associated with a 2.11% increase in GDP Regression 2: a 1 unit increase in the democracy index is associated with a 2.14% increase in GDP Regression 3: a 1 unit increase in the democracy index is associated with a .60% increase in GDP Regression 4: a 1 unit increase in the democracy index is associated with a .162% increase in GDP These results are very interesting, it appears that there is string evidence of omitted variable bias, particularly with the large drop we see in anticipated GDP from Regression 2 to Regression 3 where it would be an omitted variable upward bias which means we have been over-anticipating the results on Real GDP for the democratic index. This leads me to believe that regression 3 and 4 are likely much more accurately predict the outcome democracy index has on GDP, especially with the large increase in R2 values where in regression 3 76% of GDP increase is accounted for by the democracy index while holding all 3 variables constant and Regression 4 has almost 83% accounted for.

Since Regression 4 seems to do the best job so far lets look at how 2 ficticious countries would compare, if we held all variables constant and only looked at the extremes of the democracy index what would the change on real gdp per capita be?
